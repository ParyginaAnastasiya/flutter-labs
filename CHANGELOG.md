## Версия 1.0.0

### Основные функции
- Отслеживание настроения с выбором из 5 эмоций
- Добавление заметок к записям настроения
- Просмотр статистики и графика
- Локальное хранение данных в SQLite
- Мотивирующие цитаты из внешнего API

### Технические особенности
- Flutter 3.0+
- SQLite для локального хранения
- HTTP запросы к Forismatic API
